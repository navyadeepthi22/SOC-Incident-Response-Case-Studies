# SOC Incident Response Case Studies

This repository contains hands-on SOC alert triage and incident response case studies created to demonstrate real-world SOC analyst workflows.

## Case Studies Included
1. Potential Data Exfiltration via Zoom (False Positive)
2. Suspicious File Download from GitHub (False Positive)
3. Brute Force Login Attempts (True Positive – Escalated)

## Skills Demonstrated
- SOC alert triage
- Incident investigation and analysis
- False positive identification
- Escalation handling
- MITRE ATT&CK mapping
- Security documentation

## Purpose
This project was created as part of SOC analyst learning to showcase incident response thinking, decision-making, and documentation skills.
